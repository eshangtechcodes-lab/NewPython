## 遇到的问题
1. **迁移脚本 GBK 编码错误**: `T_COMMODITY` 表含特殊 Unicode 字符 `\ue771`，导致 `UnicodeEncodeError`。通过创建独立迁移脚本绕过。
2. **原 C# API 超时**: 部分接口响应需 30-60 秒，需加大 timeout。
3. **Router 返回结构不匹配**: `GetServerpartDDL` 和 `GetServerpartTree` 原本直接返回数组，实际原 C# 返回嵌套 `JsonList` 结构，已修正。
4. **省份编码映射问题**: `_get_province_id_by_code` 辅助函数在 API 运行时通过 JOIN/子查询方式返回空值，改为 SQL 子查询嵌入 WHERE 条件解决。
5. **旧进程占用端口**: 未完全终止的旧 Python 进程持续监听 8080 端口，导致新代码未被请求命中。需先 `taskkill` 旧进程。
