## ⚠️ 踩坑记录

### 坑 1: 大文件替换残留代码
- **现象**：`replace_file_content` 对 7000+ 行大文件操作时只替换到部分前缀，旧代码残留
- **解决**：分两步操作 — 先插入新代码，再用 Python `lines[:start]+lines[end:]` 精确删除残留行

### 坑 2: GetSummaryRevenueMonth 嵌套调用
- **现象**：该路由内部调用另一个 Router 端点 `get_summary_revenue`
- **解决**：Service 层返回 `RevenuePushModel=None`，Router 层保留嵌套调用 `revenue_push_service.get_summary_revenue()`

### 坑 3: PowerShell 多行 Python 命令
- **现象**：含 f-string + 三引号的 Python 代码在 PowerShell `-c` 中无法执行
- **解决**：改用脚本文件 (`/tmp/xxx.py`)
